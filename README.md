# HabitAI

HabitAI é uma aplicação web full stack desenvolvida para ajudar usuários a criar, acompanhar e melhorar hábitos de forma consistente. A plataforma combina gerenciamento de hábitos, estatísticas de desempenho, gamificação e recursos de Inteligência Artificial para auxiliar na construção de rotinas mais produtivas.

## ✨ Funcionalidades

✔️ Autenticação segura com JWT

✔️ Cadastro e login de usuários

✔️ Criação e gerenciamento de hábitos

✔️ Check-in diário de hábitos

✔️ Dashboard com métricas e estatísticas

✔️ Sistema de sequências (streaks)

✔️ Histórico de progresso

✔️ Gráficos de desempenho semanal e mensal

✔️ Sistema de gamificação com níveis e conquistas

✔️ Perfil do usuário e preferências

✔️ Recomendações inteligentes baseadas em IA (em desenvolvimento)

✔️ Assistente virtual para análise de hábitos (em desenvolvimento)

## 📊 Funcionalidades Principais

### Dashboard

Visualize rapidamente:

* Hábitos concluídos hoje
* Sequência atual
* Melhor sequência
* Taxa de conclusão
* Evolução semanal
* Evolução mensal

### Gerenciamento de Hábitos

Crie hábitos personalizados com:

* Nome
* Descrição
* Categoria
* Frequência
* Meta diária
* Cor personalizada

### Estatísticas

Acompanhe seu progresso através de:

* Gráficos semanais
* Gráficos mensais
* Histórico de execução
* Taxa de sucesso
* Consistência por hábito

### Gamificação

Ganhe experiência ao completar hábitos:

* Sistema de XP
* Níveis
* Conquistas
* Metas pessoais
